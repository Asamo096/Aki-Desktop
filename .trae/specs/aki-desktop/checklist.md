# Aki Desktop - Verification Checklist

- [x] 检查项目骨架搭建是否完成，目录结构是否符合设计规范
- [x] 检查 wlroots 集成是否成功，能否显示纯色背景
- [x] 检查日志系统是否正常工作，是否输出带时间戳的初始化流程
- [x] 检查信号处理是否正常，Ctrl+C 或 SIGTERM 是否能正常清理
- [x] 检查输出管理是否正常，插入显示器是否能自动识别分辨率
- [x] 检查输入处理是否正常，`Esc` 退出和 `Mod+Enter` 开终端是否工作
- [x] 检查输入延迟是否 < 8ms
- [x] 检查窗口管理是否正常，能否运行 `foot` 或 `alacritty`
- [x] 检查 BSP 平铺布局是否正常，开多个终端是否自动排列
- [x] 检查配置系统是否正常，`Mod+R` 快捷键是否能重新加载 Lua 配置
- [x] 检查渲染系统是否正常，所有窗口是否有圆角效果
- [x] 检查渲染性能是否达标，120Hz 下 99% 帧是否 < 8.3ms
- [x] 检查插件系统是否正常，插件崩溃后主进程是否存活
- [x] 检查插件是否能自动重启
- [x] 检查系统是否能正常启动和退出，无崩溃
- [ ] 检查主题系统是否正常，切换主题是否无需重启
- [ ] 检查多显示器配置是否正常，窗口拖到另一屏是否保持模式
- [ ] 检查边缘手势是否正常，鼠标拖至左边缘是否能打开应用启动器
- [ ] 检查毛玻璃效果是否正常，全局/局部模式是否可切换
- [ ] 检查 Shader 壁纸是否正常，保存 `.glsl` 文件是否即时生效
